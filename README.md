# Advanced Todo React App

A modern Todo application built with React that allows users to manage daily tasks efficiently.

## Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Delete all tasks
- 🔍 Search tasks instantly
- ✔️ Mark tasks as completed
- 💾 Automatic Local Storage persistence

## Built With

- React
- JavaScript (ES6+)
- Bootstrap
- CSS3
- Local Storage API

## Installation

1. Clone the repository

```bash
git clone https://github.com/setareh-kazemi10/react-todo-app.git
```

2. Navigate to the project folder

```bash
cd react-todo-app
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```
